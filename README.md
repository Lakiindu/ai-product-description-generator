# 🧠 AI Product Description Generator

A modern SaaS-style web app that generates professional product descriptions using AI.

## 🚀 Features
- ✨ Generate product descriptions instantly
- 📋 Copy to clipboard
- 🕘 History tracking (local storage)
- 🌙 Dark / Light mode
- ⚡ Fast and responsive UI

## 🛠 Tech Stack
- Next.js / React
- Tailwind CSS
- OpenAI API (or mock AI)
